# **App Name**: Minar Go PWA

## Core Features:

- User Authentication: Secure user registration, login, logout, and session persistence to manage foundation data.
- Member Management: Maintain a directory of foundation members, with functionality to add, view, and delete member profiles.
- Deposit & Expense Entry: Record and categorize financial transactions (deposits, Zakat, Fitra) by member with amount and date.
- Collection Summary & Filtering: Display a dynamic summary of all transactions, allowing filtering by month and showing total amounts, with option to delete entries.
- PDF Summary Report Generator: Generate professional PDF reports of filtered collection summaries for record-keeping and distribution.
- AI-Powered Demand Letter Draft Assistant: A generative AI tool to assist in drafting subject lines and body content for official demand letters based on user-provided keywords.
- Foundation Branding & Document Storage: Upload and manage the organization's logo, and store relevant documents locally within the application.

## Style Guidelines:

- Primary color: A deep, professional blue (#002366) to convey trust and stability.
- Background color: A very light, subtle grayish-blue (#F0F2F5) that is calming and keeps the focus on content.
- Accent color: A distinguished gold (#D4AF37) for highlights, important call-to-actions, and branding elements.
- System colors: A success green (#2D6A4F) for positive feedback and an error red (#DC2626) for alerts, ensuring clear communication of status.
- Headline and general text font: 'Inter' (sans-serif) for its modern, clear, and objective readability across interfaces. Note: currently only Google Fonts are supported.
- Bengali text font: 'Noto Sans Bengali' (sans-serif) specifically for all Bengali content, ensuring correct display and readability. Note: currently only Google Fonts are supported.
- Leverage the comprehensive set of Font Awesome 6.5.1 icons for a unified and visually appealing representation of actions and features.
- Mobile-first responsive design, optimized for containers up to 500px width, with adaptive grids for cards and member lists (2 columns on mobile, 4 on desktop).
- Sticky header for consistent branding and navigation, centered cards for forms like login/signup, and tables with horizontal overflow on small screens.
- Interactive elements will include touch-friendly buttons with a subtle `:active` scale effect and bottom-center toast notifications with a smooth slide-in animation.
